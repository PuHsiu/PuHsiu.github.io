從今天開始寫網誌

回想年初的時候，曾經為自己設下了幾個目標。

其中之一是這樣的：有鑑於至今都只是把學到的東西隨便記下，一直缺乏有系統的管理，尤其聽了曹祖聖老師的在2015 微軟實戰課程日的演講微軟技術無所不通-教你成為 IT 界的德魯伊後，深覺自己的知識過於零散，因此決定開始這個技術網誌。

目前計劃以 松本行弘談程式世界的未來 這本書為主軸， 閒扯 一些平日 Coding 上想法或類似的概念……之類的 XD。

仔細想想上一次寫網誌已經是高中時候的事了，那時無名小站正紅，也就是所謂的 Web 2.0 時代 XD。不過當時分享的內容主要以日常生活相關，還有一些碎碎念之類的，這次要從自己的專業出發，一邊統整知識，一邊發揮宅分享的精神 XD。

最後的一點小抱負是，希望這個網誌可以持續下去 XD，讓自己的年度生活烤雞考蹟可以漂亮一點。
