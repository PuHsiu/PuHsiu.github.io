實際操作一次

產生一個專用的 key

產生了也用不到 跳過 XD

## 產生一個實例 ```bash boot \ --flavor FLAVOR_NAME \ --image IMAGE_NAME \ --nic net-id=NET_ID \ --security-group default \ --key-name KEY_NAME \ INSTANCE_NAME ```
## 綁定 Floating IP ``` neutron floatingip-create EXTERNAL_NETWORK_ID nova floating-ip-associate INSTANCE_NAME NEW_GENERATE_IP ```
## 外掛虛擬硬碟 ``` cinder create --display-name VOLUME_NAME SIZE(GB) nova volume-list nova volume-attach INSTANCE_NAME VOLUME_ID ```
## 最後檢驗一下 ``` nova list nova volume-list ```
## 大功告成！ ``` nova get-vnc-console INSTANCE_NAME novnc ```
